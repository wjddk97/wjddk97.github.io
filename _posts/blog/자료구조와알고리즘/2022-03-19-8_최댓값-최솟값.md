## 최댓값 찾기

---

```python
class MaxAlgorithm:

    def __init__(self, nums):
        self.nums = nums
        self.maxNum = 0

    def getMaxNum(self):
        self.maxNum = self.nums[0]

        for n in self.nums:
            if self.maxNum < n:
                self.maxNum = n

        return self.maxNum

nums   = MaxAlgorithm([-2, 4, 7 ,-6, 11, 5])
maxNum = nums.getMaxNum()
print(f'maxNum: {maxNum}')

-----
maxNum: 11
```
