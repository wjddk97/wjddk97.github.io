- 2번째 시도 (통과)

```python
def solution(participant, completion):
    participant.sort() #선수 
    completion.sort()  #완주자
    completion.append(0)

    for p, c in zip(participant, completion):
        if p != c:
            return p
```
1. 선수, 완주자 모두 정렬을 해준다.
2. 길이를 맞추기 위해 완주자 리스트 마지막에 요소 추가
3. zip()을 이용해 선수와 완주자를 동시에 for문에 돌린다.
4. 선수p 와 완주자c 가 다를 경우 선수p반환
